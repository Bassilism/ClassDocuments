ชี้แจง : โจทย์ข้อนี้ไม่ได้เป็นการสปอยหนัง TENET แต่จะเป็นการนำเอา Concept เรื่องเวลามาใช้  
น้องที่ยังไม่ได้ดูสามารถอ่านโจทย์ได้อย่างสบายใจครับ

เนื้อเรื่อง : กฤษฎาได้รับเลือกเป็น 1 ในทีมพิทักษ์โลก โดยศัตรูในครั้งนี้เป็นศัตรูที่มาจากอวกาศ 
ทีมพิทักษ์โลกได้เลือกใช้วิธีการจู่โจมแบบขนาบข้างเพราะมีอัตราชนะที่สูงกว่า โดยการจู่โจมแบบขนาบข้างนั้น
จะแบ่งเป็น 2 ทีมคือ ทีมสีแดง และ ทีมสีน้ำเงิน  โดยที่ทีมสีแดงนั้นจะเป็น Timeline ปัจจุบัน  
ส่วนทีมสีน้ำเงินนั้นจะเป็น Timeline จากอนาคต  โดยทีมสีน้ำเงินจะเห็นสิ่งที่เกิดขึ้นกับทีมสีแดง 
ทำให้ทีมสีน้ำเงินเข้าเครื่องหมุน เพื่อกลับมาเปลี่ยนเหตุการณ์ที่จะเกิดขึ้นกับทีมสีแดงได้  
ซึ่งสิ่งที่ทีมสีน้ำเงินเจอจะเป็นเอนโทรปีย้อนกลับ  ซึ่งในโจทย์นี้จะเกี่ยวกับการระเบิด 
ซึ่งถ้าหากเกิดการระเบิดกับทีมสีแดงระเบิดก็จะเป็นแบบปกติ แต่ถ้าหากเกิดกับทีมสีน้ำเงินระเบิด
จะกลายเป็นน้ำแข็งแทน  ซึ่งทีมสีน้ำเงินจะนำระเบิดที่เป็นน้ำแข็งไปขัดขวางการระเบิดของทีมสีแดงได้  
โดยการระเบิดเกิดจากการเรียงตัวอักษร 

อธิบายรูปแบบ Input ของ Test_Case_1 : ทีมสีแดงจะมีระเบิดเรียงดังนี้ -> AAABBBCDEE  
ทีมสีน้ำเงินจะมีระเบิดเรียงดังนี้ -> HHH โดยทีมสีน้ำเงินจะมีระเบิด H ที่เป็นน้ำแข็ง 1 ลูกไว้สำหรับป้องกัน
การระเบิดที่จะเกิดทีมสีแดงได้  ต่อมาทีมสีแดงจะเกิดการระเบิดของ A และ B ตามลำดับ  
โดยทีมสีน้ำเงินจะนำระเบิด H ไปขัดการระเบิดของระเบิด A เพราะระเบิด A เกิดการระเบิดก่อนระเบิด B  
โดยการขัดระเบิดนั้นจะเป็นการขัดระหว่างลูกที่ 2 กับลูกที่ 3 
เพื่อให้เห็นภาพ -> AAABBBCDEE -> AA(H)ABBBCDEE  -> AA(H)ACDEE ลำดับจะเป็นดังนี้  
และทีมสีแดงเกิดการระเบิด 1 ครั้ง ส่วนทีมสีน้ำเงินก็เกิดการระเบิดอีก 1 ครั้ง

อธิบายรูปแบบ Input ของ Test_Case_3 : ทีมสีแดงจะมีระเบิดเรียงดังนี้ -> AAABBBCDDDEE  
ทีมสีน้ำเงินจะมีระเบิดเรียงดังนี้ -> BBBTENETAAA โดยทีมสีน้ำเงินจะมีระเบิด A และ B ที่เป็นน้ำแข็ง 2 ลูกตามลำดับ
ไว้สำหรับป้องกันการระเบิดที่จะเกิดทีมสีแดงได้  ต่อมาทีมสีแดงจะเกิดการระเบิดของ A B และ D ตามลำดับ  
โดยทีมสีน้ำเงินจะนำระเบิด A  ไปขัดการระเบิดของระเบิด A เพราะระเบิด A เกิดการระเบิดก่อนระเบิด B  
โดยการขัดระเบิดนั้นจะเป็นการขัดระหว่างลูกที่ 2 กับลูกที่ 3  
เพื่อให้เห็นภาพ -> AAABBBCDDDEE -> AA(A)ABBBCDDDEE -> ABBBCDDDEE ลำดับจะเป็นดังนี้  
ต่อมาจะนำระเบิด B ไปขัดการระเบิดของระเบิด B 
เพื่อให้เห็นภาพ  ABBBCDDDEE -> ABB(B)BCDDDEE -> ABCDDDEE  
ต่อมาเกิดการระเบิดอีก 1 ครั้ง ABCDDDEE -> ABCEE ซึ่งสีน้ำเงินไม่สามารถขัดได้เพราะระเบิดน้ำแข็งหมดแล้ว   
และทีมสีแดงเกิดการระเบิดทั้งหมด 3 ครั้ง  ซึ่ง 2 ครั้งเกิดจากการที่ทีมสีน้ำเงินใส่ระเบิดสีเดียวกันมาซึ่งถือว่า
เป็นความผิดพลาดและเกิดการระเบิดเองอีก 1 ครั้ง ส่วนทีมสีน้ำเงินก็เกิดการระเบิดอีก 2 ครั้ง
